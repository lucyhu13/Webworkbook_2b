Week 7

Toolkit

This week's class we went to Telstra creative space for class. This week's course impressed me deeply and gave me a deeper understanding of programming. We first learned about the parts in the tool kit, including their specific functions and how to work together. For example, the most important thing is the control board, which is responsible for connecting and controlling other parts, and is used to receive and output instructions.

Arduino

We also learned how to use this software, which is an open source electronic prototyping platform. Development tools for writing, uploading, and debugging Arduino code based on C/C++. By understanding this software, I learned that eletronic and programming are essentially a lot of small parts and switches that are on or off. The principle of programming is to make the computer do something by setting different combinations of these switches, and finally present a running state.

Led blink

In Arduino, we use the blink in the example. First, we need to install the hardware connection according to the picture given by the teacher. We need Bread Board (, Jumper (wire), Push Button Switch, resister (electrical component that limits or regulates the flow of current in an electronic circuit) and LED. If the button is pressed, the LED will light up.

Buzzer

In the second activity, we learned how to make an inductive buzzer. The most important sensor is the Ultrasonic distance sensor, which senses the distance of objects by emitting ultrasonic signals.
